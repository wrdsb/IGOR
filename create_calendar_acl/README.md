# Create Calendar ACL

https://developers.google.com/google-apps/calendar/v3/reference/acl

```javascript
{
  "kind": "calendar#aclRule",
  "etag": etag,
  "id": string,
  "scope": {
    "type": string,
    "value": string
  },
  "role": string
}
```
